YaST2: Overview: Features


  • Basics
    • Full replacement for linuxrc and YaST1
    • Selectable interface (Graphic, Text, KDM, Gnome, ...)
    • Networking, remote installation and administration
    • Highly modularized
    • Easily expandable and adaptable to new systems
    • Multilingual
    • Support for non-intel based systems

  • Booting
    • Autodetection
      autodetect hardware, autoload modules
    • Configuration management
      restore running config data after misconfig
      Select configuration at boot time (home, office, ...)
    • Internationalisation
      Translation of error messages
    • Help-System
      Explain what happens, context-sensitve help
    • No console
      Hide console from user, boot to xdm/kdm

  • Installation
    • Graphical installer
      hide kernel messages from user
    • Workflow
      Guide user through neccessary steps
    • Automatic installation
      load workflow from disk/internet
    • Autodetection
      Automated hardware detection with default values and workflow
    • Networking
      Network install from server

  • Administration
    • Workflow
      Guided administration, workflow driven
    • Configuration management
      Keep data in database
      versioning
      multiple configurations selectable
    • Network administration
    • Cluster administration
