Features

Basic Features

  • Two step install boot

    • boot enhanced linuxrc with hw autodetection
    mount live-filesystem automatically
    start X11 (Qt) from live filesystem

  • Selectable interface

    • Curses based or Framebuffer/Qt based
    Common look & feel
    Useable via network
    Freely programmable
    Multilingual

  • Highly modularized

    • Separate interface, workflow and hardware layer
    Building block principle, parts easily exchangeable
    Easily expandable and portable to new systems

  • Automatic installer

    • autodetect hardware, autoload modules
    automatic partitioning of harddisk
    install selectable (but predefined) set of rpm-pakets

  • Configuration

    • Offer configurations for each detected peripherial
    Workflow driven configuration
    Use SaX for X11 configuration
    Offer internet, sound, and printer configuration at minimum

Advanced features

  • loadable workflows

    • provide preconfigured workflows via ftp
    allow record/playback of workflows

  • multiple configurations

    • support multiple configurations, selectable at boot

  • configuration versioning

    • add version management to config files
    allow 'back to working config' after mis-configuration
    fallback to safe boot-environment

  • network interface

    • remote installation and adiminstration

  • network repository

    • single database for larger pool of computers
