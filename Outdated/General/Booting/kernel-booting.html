YaST2: Analysis: How Linux boots

This document describes how linux boots.

On Intel Hardware

  1. load kernel into memory (LILO or LOADLIN)
  2. load ramdisk image into memory (LILO or LOADLIN)
  3. initialize kernel, parse arguments (init/main.c)
  4. start /linuxrc if initial ramdisk as root
  5. change root device (specified in kernel parameter)
  6. if 'init=' set, start specified program
  7. else try /sbin/init
  8. else try /etc/init
  9. else try /bin/init
  10. else try /bin/sh
  11. else panic ()

... On Alpha Hardware

On Alpha PC (aka Jensen)

  1. BIOS reads start sector and size of boot image from boot sector
  2. BIOS reads boot image
  3. BIOS starts boot image (MILO)

... On Sparc Hardware

  1. BIOS (OpenPROM) reads boot sector
  2. BIOS reads boot image from disk
  3. BIOS starts boot image (SILO)
