YaST2: Analysis: Linuxrc: Purpose and Tasks

This document describes the purpose and task of linuxrc for SuSE Linux

From the README file 

Linuxrc is a program used for setting up the kernel for installation purposes.
It allows the user to load modules, start an installed system, a rescue
system or an installation via YaST.

Linuxrc is designed to be as small as possible. Therefore, all needed programs
are linked directly into one binary.

The following programs are used (i.e. built-in)

bootpcBOOTP Client
insmodInstallation of Kernel Modules
loadkeys
cardmgrPCMCIA card manager
inflateDecompress images
linuxThe Kernel
