YaST2: Analysis: Linuxrc: Calling YaST

This is a short description how yast is called by linuxrc -- the exact procedure calls can be referred within source code.

Call sequence

Following call sequence is defined within install.c: 

inst_start_install()            inst_auto_install()
        |    (with menue)              |
        |                              |
        +-------------+----------------+
                      |
                      V
              inst_execute_yast()
                      |
                      V
              system( /sbin/YaST ... )

What happens calling linuxrc:

  • the passed parameters are evaluated
  • linuxrc is initialized
  • inst_auto_install() is called
  • linuxrc is terminated

Within inst_auto_install() following steps are performed:

  • inst_choose_source() or if source is known inst_mount_*()
  • root_load_rootimage()
  • then the image is mounted
  • and YaST is called
