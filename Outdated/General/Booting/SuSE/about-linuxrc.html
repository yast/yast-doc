YaST2: Design: Tasks: linuxrc

This documents describes linuxrc, the program for configuring a running kernel and to mount installation source and destination filesystems.

What is linuxrc ?

Linuxrc is a special program which is started instead of /sbin/init if the root filesystem resides on the initial ramdisk. This behaviour is hard coded into the linux kernel und usually used for rescue disks.

Further documentation can be found in the Initial Ramdisk Howto in the Documentation directory of the kernel sources.

Purpose of linuxrc

Linuxrc is used to prepare a running system for further tasks. Usually this means
    loading modules
    mounting filesystems
    starting other programs (i.e. /sbin/init)
linuxrc can be used for anything that can be done in a standalone, single-tasking system without defined environment.

linuxrc in SuSE linux is solely used as a 'boot helper' on a rescue disk or at initial boot, when no linux is installed.

In the YaST2 environment (i.e. automatic installer) linuxrc's purpose is defined by the first steps of the installer up to mounting a live filesystem.
