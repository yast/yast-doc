Deprecated WFM builtins

Instead ofuse
General shell commands
Shell (command, env) Execute (.local.bash, command, env)
Removing a file
Shell ("/bin/rm file") Execute (.local.remove, file)
Creating a directory
MkDir (directory) Execute (.local.mkdir, directory)
Removing a directory
Shell ("/bin/rmdir directory") Execute (.local.rmdir, directory)
Mounting/Unmounting
Shell ("/sbin/mount device directory") Execute (.local.mount, [device, directory , optional-log] , mount-options)
Shell ("/sbin/umount directory)" Execute (.local.umount, directory )
Reading and writing text files
WriteString (filename, value) Write (.local.string, filename, value)
ReadString (filename) Read (.local.string, filename)
Reading and writing YCP files
Write (filename, value) Write (.local.ycp, filename, value)
WriteY2 (filename, value) Write (.local.yast2, filename, value)
WriteY2 (filename, value, mode) Write (.local.yast2, [filename, mode ], value )
Read (filename) Read (.local.ycp, filename)
ReadY2 (filename) Read (.local.yast2, filename)
