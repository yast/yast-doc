YaST2 Developers pages: Compiling

If you want to compile the complete C++ source code (usually not needed), you need the following packages installed:

  • qtlib2
  • qtdevel2
  • gperf
  • automake
  • autoconf
  • ncurses
  • flex
  • bison
  • egcs
  • gcc
  • gpp
  • gppshare
  • libgpp
  • gettext
  • xdevel
For testing (via make check) you'll also need
  • tcl
  • tk
  • expect
  • dejagnu
After checking out (with cvs), cd to a source directory and execute (as non-root): 
make -f Makefile.cvs
make
and as root 
make install
If you checked out the complete yast2/source tree, there is a makeall shell script which compiles everything in the right order.
