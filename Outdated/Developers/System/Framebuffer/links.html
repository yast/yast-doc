YaST2: Resources: Framebuffer: Links

The framebuffer device /dev/fb0 allows generic access to all kinds of framebuffer (pixel based display) devices across multiple platforms. It´s integrated into the kernel (2.2.x) and there is also a X11 device driver (XFBdev) for using it under XFree86.
